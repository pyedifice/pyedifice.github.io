edifice.Image

edifice.Image#

class edifice.Image(src='', scale_to_fit=True, **kwargs)[source]#

Bases: QtWidgetComponent

An image container.

Parameters:

  • src (Any) – either the path to the image, or an RGB np array, or a QtGui.QImage. The np array must be 3 dimensional (height, width, channels)

  • scale_to_fit (bool) – if True, the image will be scaled to fit inside the container.
