edifice.Image

edifice.Image#

class edifice.Image(src, aspect_ratio_mode=None, **kwargs)[source]#

Bases: QtWidgetElement

An image container.

  • Underlying Qt Widget QLabel

If you want to display a 3-dimensional uint8 numpy array as an image, checkout:

  • https://pyedifice.github.io/stubs/edifice.extra.NumpyImage.html

  • https://pyedifice.github.io/stubs/edifice.extra.NumpyArray_to_QImage.html

Parameters:

  • src (str | QImage | QPixmap) –

    One of:

    • A path to an image file.

    • A QImage.

    • A QPixmap.

  • aspect_ratio_mode (Optional[AspectRatioMode]) –

    The aspect ratio mode of the image.

    • None for a fixed image which doesn’t scale.

    • An AspectRatioMode to specify how the image should scale.
